Help

The following information gives an overview of how the e-learning is structured and how to navigate through the screens, module(s) and course.

Basic Navigation

When you see this button in the main body of the e-learning , press it to move to the next section of information . This can display further information within a screen, else the next screen in the training sequence.

Navigation using the Menu

Once you have visited a screen it will appear in the menu tab. You can use this tab to navigate to any screens you have already visited within a module.

You may access this tab by moving your mouse to the bottom of the screen.

Completing the Course

The Progress Bar tells you how far through the course you are.

Other Functions

The volume can be turned on/off using the mute button .
